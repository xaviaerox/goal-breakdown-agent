# Timeline Estimator Skill

Based on planning outputs and user constraints:
1. Estimate total hours needed for the goal.
2. Compare required hours with weekly availability windows.
3. Calculate target completion weeks/months and match with the deadline.
4. Output the estimated duration and pacing validation (e.g. "4 weeks - achievable").
